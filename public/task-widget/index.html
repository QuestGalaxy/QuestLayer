FAQ

How is a task widget different from a quest widget?

Task widgets focus on specific actions, while quest widgets group tasks into a broader mission flow.

Can tasks link out to external destinations?

Yes. Each task can open a link, trigger an integration, or collect proof.

Do users need an account to complete tasks?

No. You can start with anonymous tasks and add wallet login when needed.

Can I change task order or rewards later?

Yes. Edit tasks in the dashboard and updates appear immediately in the widget.
